Проект ВКР по теме прогнозирования качества сна на основе данных носимых устройств.

## Стек
- Python
- pandas
- scikit-learn
- CatBoost
- Jupyter Notebook
- VS Code

## Структура
- `data/` — данные
- `notebooks/` — исследовательские ноутбуки
- `src/` — основной код
- `models/` — сохраненные модели
- `reports/` — отчеты и метрики
- `figures/` — графики
